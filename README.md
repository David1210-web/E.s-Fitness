# E.S Fitness Website 🏋️

A modern, responsive fitness website built for a gym brand to showcase services, trainers, membership plans, and contact information.

## 🚀 Live Demo

You can click it here
(https://e-s-fitness.vercel.app/)

## 📌 Features

- Responsive design for desktop, tablet, and mobile
- Modern gym landing page
- Hero section with call-to-action
- Services section
- Trainer showcase
- Pricing/membership plans
- Contact section
- Smooth animations and interactive elements

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- Git & GitHub
- Vercel Deployment

- 
## 🎯 Purpose

This project was created as a portfolio project to demonstrate modern website development skills and responsive business website design.
